Description

This is a community project started by the modders at the I-War 2 Discord Server designed to fix bugs that exist in the final F14.6 version of the game. This version of the mod only fixes those bugs, and does not add any quality of life improvements to the game.

List of current bug fixes:

Dagda crash while playing in the system or doing “Corporate Holdings”

“Wolf’s Den” freighter hangup

Ships stopping at the L-point, or not jumping with their convoy
